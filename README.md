# Зельеварня 🛸

Браузерная игра про воссоздание смесей по памяти. Лавка на краю вселенной.

## Структура проекта

| Файл | Что внутри | Когда трогать |
|---|---|---|
| `index.html` | разметка-оболочка | почти никогда |
| `style.css` | весь визуал + гид "куда вставлять картинки" (в шапке файла) | перекраска, рамки, фоны |
| `content.js` | пришельцы, фразы, формы, сложности, настройки | **постоянно** — весь новый контент сюда |
| `game.js` | логика игры | только новые механики |

Все 4 файла кладутся в корень репозитория рядом. Никакой сборки не нужно.

## Как добавить нового пришельца / фразы

Открой `content.js` → массив `DIFFICULTIES` (или `SPECIAL_ORDERS`). Фразы — просто строки в `flavors` (и `ff` для фокус-заказов), дописывай сколько хочешь. Новый НПС — скопируй существующий блок и поменяй поля.

## Как заменить эмодзи на свои картинки

1. Создай в репозитории папку `assets/` (при загрузке файла впиши путь `assets/npc/drone.png` — папки создадутся сами).
2. PNG с прозрачным фоном, ~256×256 для портретов.
3. В `content.js` добавь НПС поле `img: 'assets/npc/drone.png'` — картинка заменит эмодзи. Стикеры результата: пути в `STICKERS`.
4. Рамки, фоны, циферблат, кнопки — инструкция в шапке `style.css`.

Можно заменять постепенно: у кого нет `img`, тот остаётся на эмодзи.

## GitHub Pages

Залей все файлы в корень публичного репозитория → Settings → Pages → Deploy from a branch → main / (root). Ссылка: `https://твой-ник.github.io/имя-репо/`.

**Важно про кодировку:** редактируй файлы через веб-редактор GitHub (карандашик) или редактор с UTF-8 (VS Code, Notepad++). Блокнот Windows может сломать русский текст.

## Глобальный рейтинг

В `content.js`, в самом верху:
```js
const CONFIG = {
  FIREBASE_DB_URL: 'https://твой-проект-default-rtdb.firebaseio.com',
};
```
Как завести бесплатную базу Firebase (5 минут): console.firebase.google.com → Add project → Build → Realtime Database → Create Database → Start in test mode → скопировать URL. Без URL рейтинг работает локально (только этот браузер).
